## 📁 Using Cheat Tables

Cheat tables provide a simple way to activate changes without searching for memory addresses manually.

1. Open the game you want to modify.
2. Open Cheat Engine.
3. Click the computer icon in the top left corner of the window.
4. Select your game process from the list.
5. Click File in the menu bar.
6. Choose Load and select your cheat table file (.ct).
7. Select the boxes next to the options at the bottom of the window to activate them.

## 💡 Frequently Asked Questions

What games work with this?
This tool works with many single-player games. It does not work with online multiplayer games because those games use servers to track your data.

What is a cheat table?
A cheat table is a file that stores specific memory locations for a game. It allows you to toggle features like unlimited health or ammo.

Is this safe?
Use this tool only for single-player games. Do not use it for online games as this might result in a ban from those services.

How do I find a value?
Enter the current value you see in your game (like your gold count) into the value box. Click First Scan. Change your gold count in the game. Enter the new value into the box. Click Next Scan. Repeat this process until you find the right address.

Can I change game speed?
Yes. Open the speedhack option in the settings menu. Enter the multiplier you want for your game speed and click Apply.

What happens if the game crashes?
Some memory changes might cause a game to crash. Always save your game progress before you make changes. 

Do I need internet access?
You need an internet connection to download the tool. Once installed, you can use the software offline.

How do I create a trainer?
Use the Trainer Maker tool within the application. It allows you to package your changes into a standalone file that acts as a trainer for others.

Does it support game mods?
It integrates well with various game modding tools. You can use it alongside other mods to enhance your experience in single-player titles.
